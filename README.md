# MealGenius

## Présentation

MealGenius est une application mobile qui permet de trouver des repas en fonction des ingrédients que vous possédez. Vous pouvez également ajouter des recettes à vos favoris et les consulter à tout moment. Chaque utilisateur possède un compte unique et personnel, il peut donc ajouter des ingrédients à son garde-manger et les supprimer à tout moment.

## Installation
- Cloner le projet
- Installer les dépendances avec `npm install`
- Installer Expo avec `npm install --global expo-cli`
- Lancer le projet avec :
  - `expo start` pour lancer le projet sur un réseau local
  - `expo start --tunnel` pour lancer le projet sur un réseau externe
- Scanner le QR code avec l'application Expo Go sur votre téléphone
- Enjoy :smile: !

## Fonctionnalités
- Theme Sombre / Clair
- Ajout d'ingrédients au garde-manger
- Suppression d'ingrédients du garde-manger
- Ajout de recettes aux favoris
- Suppression de recettes des favoris
- Visualisation des recettes avec les ingrédients et les étapes de préparation
- Proposer des recettes en fonction des ingrédients du garde-manger
- Proposition des recettes journalières
